# General professional writing

## Contract

- Audience: identify the real reader, not “the public”.
- Purpose: inform, decide, approve, explain, persuade, or record.
- Default register: clear, restrained, specific.

## Review priorities

1. The main point appears before background that only the author needs.
2. Facts and judgments are distinguishable.
3. Each paragraph has one primary function.
4. Claims name actors, actions, scope, and time where relevant.
5. The ending completes the document rather than handing the conversation back to the user.

## Protected features

Preserve facts, numbers, terms, citations, uncertainty, required headings, and the author's stable choices.

## Common failures

- task recap instead of content;
- generic importance claims;
- repeated summary sentences;
- excessive headings or bold-first bullets;
- chat greetings and sign-offs in a formal artifact.
