# ReviewWrite 分发包

这里是由正式版本自动生成的对外分发素材，不是运行时 Skill 内容。

每次创建 `vX.Y.Z` 标签后，GitHub Actions 会生成一个 `distribution-kit-X.Y.Z` 压缩包，包含：

- 各平台可直接使用的一句话安装指令；
- GitHub、Gitee、SkillHub、Agent Skills 目录和 ClawHub 的简介；
- 官方仓库、镜像、Release、案例和校验信息；
- 适合公众号、朋友圈、视频号的短文案。

分发原则：GitHub Release 是版本与校验的主来源，Gitee 只作同步镜像；第三方目录只提交同一版本的素材，不在平台之间维护不同的 Skill 内容。没有平台 API 或账号授权时，流程只生成提交包，不猜测接口、不写入账号。
